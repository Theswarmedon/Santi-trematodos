# Trematodos - Clase Trematoda

Este proyecto es una página web educativa sobre los trematodos (clase Trematoda), un grupo de gusanos platelmintos parásitos.

## Contenido

La página incluye información sobre:
- Introducción a los trematodos
- Características morfológicas y fisiológicas
- Taxonomía y clasificación
- Ciclo de vida complejo
- Importancia sanitaria y enfermedades relacionadas
